# MovieRate (Python)

## 🇬🇧 English

This is a Python desktop application that recommends movies using a neural network trained with TensorFlow.

### Description

The application generates personalized movie recommendations based on the user's selected favorite movies. Recommendations are produced using a pre-trained TensorFlow/Keras model through a desktop interface built with Tkinter.

### Features

- desktop application with a Tkinter interface
- personalized movie recommendations
- pre-trained neural network model
- movie data processing with Pandas
- model inference using TensorFlow/Keras

### Technologies

- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Tkinter

### Dataset

The project uses the MovieLens dataset provided by GroupLens:

- https://grouplens.org/datasets/movielens/

## 🇷🇺 Русский

Это настольное приложение на Python, рекомендующее фильмы с помощью нейронной сети, обученной с использованием TensorFlow.

### Описание

Программа формирует персональные рекомендации фильмов на основе выбранных пользователем любимых фильмов. Для получения рекомендаций используется предварительно обученная модель TensorFlow/Keras, а взаимодействие с пользователем осуществляется через графический интерфейс Tkinter.

### Возможности

- приложение с графическим интерфейсом Tkinter
- персонализированные рекомендации фильмов
- обработка данных с помощью Pandas
- применение обученной нейронной сети для генерации рекомендаций

### Технологии

- Python
- TensorFlow
- Keras
- Pandas
- NumPy
- Tkinter

### Данные

Используется датасет MovieLens от GroupLens:

- https://grouplens.org/datasets/movielens/
